ARC Study
Clinical Records

An interactive viewer for CCDA documents and FHIR data from ARC Study participants. Includes structured codes (ICD, RxNorm, CPT, LOINC, SNOMED), clinical narratives, and 13 FHIR resource categories. All data stays local — nothing is uploaded.

Pipeline:

  1. ccda_parser_v4.py — extract from decoded CCDA files
  2. fhir_parser.py — extract from fhir_assembled JSON files
  3. final_joiner_v3.py — merge CCDA + FHIR by patient_id
  4. enrich_display_names.py — fill in LOINC labels (optional)
  5. Click ▲ Load → choose dashboard_data.json
